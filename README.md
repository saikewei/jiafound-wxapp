# 嘉园拾遗 JiaFound

---

## 如何用 VS Code 配置并运行本项目（UniApp 微信小程序）

1. **安装依赖**

   - 打开终端，执行：
     ```sh
     pnpm install
     ```

2. **本地开发预览**

   - 运行：
     ```sh
     pnpm dev:mp-weixin
     ```
   - 生成 `dist/dev/mp-weixin` 目录。

3. **微信开发者工具预览**
   - 安装微信开发者工具
   - 用微信开发者工具导入`dist/dev/mp-weixin` 项目，选择嘉园拾遗 appid，即可预览和调试小程序。
